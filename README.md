# Análisis de Gestión de Reclamos

Herramientas utilizadas:
- Excel
- SQL / MySQL
- Power BI

Descripción:
Proyecto de análisis de datos enfocado en la gestión de reclamos,
evaluando volumen, tipos, tiempos de atención y tendencias.
El objetivo es identificar patrones y apoyar la mejora de
procesos y la toma de decisiones operativas.
